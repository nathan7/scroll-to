
# scroll-to

  scroll to an element

## Installation

    $ component install nathan7/scroll-to

  or

    $ npm install scroll-to

## API

### scrollTo(elem)
### scrollTo.y(elem)

  Scrolls to the given element in Y.

### scrollTo.x(elem)

  Scrolls to the given element in X.

### scrollTo.xy(elem)

  Scrolls to the given element in X and Y.
